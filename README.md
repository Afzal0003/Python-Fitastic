# 💪 Fitastic - Health & Fitness Tracker

Fitastic is a desktop-based Health and Fitness Tracker developed using **Python** and **Tkinter**. The application helps users monitor daily activities, maintain health records, calculate BMI, track fitness progress, and estimate calorie goals required to achieve a target weight.

---

## ✨ Features

### 🏃 Daily Activity Tracking
- Log daily step count
- Track calories burned
- Record daily water intake
- Save exercise details

### ❤️ Health Monitoring
- Record weight and height
- Automatically calculate Body Mass Index (BMI)
- Store health records for future reference

### 📊 Progress Tracking
- View complete activity history
- Monitor health records
- Display data using interactive tables
- Refresh records instantly

### 🎯 Fitness Goal Planner
- Set target weight
- Estimate daily calorie intake required to reach fitness goals
- Personalized calorie recommendations

### 💾 Persistent Data Storage
- Stores activity logs in CSV files
- Saves health measurements locally
- Automatically creates required data files

## 🚀 Features Overview

### Activity Logger
Users can record:

- Daily Steps
- Calories Burned
- Water Intake
- Exercise Details

### Health Manager

Stores:

- Weight
- Height
- BMI

BMI is automatically calculated using:

```
BMI = Weight (kg) / Height² (m²)
```

### Progress Dashboard

Displays:

- Activity History
- Health History
- Refreshable data tables

### Goal Calculator

Calculates estimated daily calorie intake required to reach a desired target weight based on current body weight and calorie deficit calculations.
